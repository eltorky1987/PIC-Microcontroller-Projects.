# ⚡ PIC Microcontroller Projects | مشاريع المتحكمات الدقيقة  

## 📖 الوصف | Description  
مجموعة مشاريع وتجارب مبنية على متحكمات **PIC**، تهدف لتعلم أساسيات البرمجة المدمجة والتحكم في الأجهزة الإلكترونية.  
تشمل أمثلة عملية على استخدام الحساسات، المحركات، والدوائر الإلكترونية.  

---

## 🛠️ المكونات | Components  
- PIC Microcontroller (مثل PIC16F877A)  
- MPLAB IDE + XC8 Compiler  
- Breadboard + أسلاك توصيل  
- حساسات (درجة حرارة، دخان، حركة)  
- محركات أو وحدات إخراج (LED, Buzzer, LCD)
